USB-C LED Power PCB

Custom USB-C sink PCB for regulated LED power

Project Overview

This project involved designing and validating a custom PCB that negotiates 5V power from USB-C sources using proper sink logic. The board provides regulated power for LEDs.

Key Features

  • USB-C Sink implementation using CC pull-down resistors
  • 5V regulated output for LEDs
  • Compact PCB designed in EasyEDA
  • Designed for modern USB-C power sources

Tools & Technologies

EasyEDA USB-C PCB Design Power Electronics Hardware Validation

Design & Validation

Hardware functionality was validated through Design Rule Checks (DRC) and hands-on testing across five manufactured units. Testing confirmed stable 5V delivery from multiple USB-C power sources.
