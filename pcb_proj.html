R2-D2 Interactive Robot

Embedded robotics project with autonomous and interactive behaviors

Completed:

2024

Platform:

Microcontroller-based robotics platform, motors, sensors, audio

Roles:

Embedded Programming, Electronics Integration, Systems Testing

Description:

This project involved designing and programming an interactive R2-D2 inspired robot capable of autonomous movement and reactive behaviors. The robot responds to sensor input and user interaction through coordinated motor control, sound playback, and state-based logic.

Key Features:

  • Autonomous and reactive motion control
  • Sensor-driven state machine
  • Integrated audio feedback
  • Modular embedded software design
