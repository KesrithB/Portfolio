Weatherboard

Animated LED weather display powered by live API data

Completed:

2024

Platform:

Arduino Uno R4 WiFi, WS2811 LEDs

Roles:

Embedded Programming, Hardware Design, Data Visualization

Description:

Weatherboard is an animated LED display that visualizes real-time weather data retrieved from an external API. JSON data is fetched, parsed, and mapped to dynamic lighting patterns across a 150-LED array, providing an intuitive and ambient weather readout.

Key Features:

  • Live weather data via external API
  • JSON deserialization on embedded hardware
  • Individually addressable LED animations
  • WiFi-enabled microcontroller
