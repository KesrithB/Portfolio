Weatherboard

Real-time weather visualization using an addressable LED array

Project Overview

Weatherboard is an animated LED display that visualizes real-time weather data retrieved from an external API. The system fetches, deserializes, and interprets JSON data, mapping weather conditions to dynamic lighting patterns across an addressable LED array.

Key Features

  • Live weather data fetched from an external API
  • JSON parsing on embedded hardware
  • Animated visualizations using WS2811 LEDs
  • WiFi-enabled microcontroller

Tools & Technologies

Arduino Uno R4 WiFi WS2811 LEDs REST API JSON Embedded Systems

Implementation Details

The system periodically retrieves weather data over WiFi and deserializes JSON responses to extract key metrics such as temperature and conditions. These values are then translated into color schemes and animation patterns, creating an intuitive and ambient weather display.
