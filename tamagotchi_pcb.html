Tamagotchi Handheld Game Console

Custom PCB for Tamagotchi Game Console

Project Overview

Led an electronics workshop of 20 students where each participant soldered and programmed a Tamagotchi game console.

Key Features

  • 2 layer pcb
  • Active low GPIO button Configuration
  • I2C communication protocol used to communicate with LDC
  • Through hole soldering
  • 3D printed housing designed in Fusion

Tools & Technologies

KiCad LCD Programming PCB Design ESP 32 Hardware Validation

Design & Validation

Ensured design passed DRC check, and that all footprints matched the selected components prior to ordering. Assembled 3 consoles to ensure the design was functional.
