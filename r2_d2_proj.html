R2-D2 Robot

Embedded robotics project with pre-programmed movement pattern

Project Overview

This project involved designing and programming a battery powered R2-D2 inspired robot capable of autonomous movement and responsive behavior.

Key Features

Tools & Technologies

Microcontroller Embedded C/C++ Motor Control Robotics

Implementation Details
