LED Climbing Training Board

Interactive climbing board with individually addressable LED guidance

Project Overview

This project involved designing and building an interactive climbing training board using individually addressable LEDs to illuminate specific holds. The system enables programmable routes and training patterns, allowing climbers to focus on technique, strength, and consistency.

Key Features

  • Individually addressable LED guidance for climbing routes
  • Custom control firmware for dynamic route patterns
  • Scalable LED array design (150+ LEDs)
  • Durable integration with a physical training board

Tools & Technologies

Arduino Addressable LEDs Embedded C++ Custom Hardware Rapid Prototyping

Implementation Details

The control system drives an array of individually addressable LEDs, allowing precise control over color and illumination patterns. Custom firmware enables route sequencing and difficulty scaling, while the hardware design ensures reliability under repeated physical use.
